Role-Based Dashboards 👥

Four unique experiences connected to one secure database.

(Swipe down to explore roles)

1. Administrator Dashboard

The Command Center

  • Global Overview: Real-time statistics on enrollments, attendance, and staff counts.
  • Financial Control: Track tuition, operational expenses, and process teacher payroll.
  • User Management: Create accounts, assign roles, and handle HR leave requests.
  • Communication Hub: Broadcast urgent announcements across the platform.

2. Teacher Dashboard

The Academic Workspace

  • Classroom Management: Interactive timetables and digital daily attendance.
  • Grading & Exams: Upload results, assign grades, and update student progress dynamically.
  • Personal HR: Track monthly pay slips and officially request days off.
  • Direct Messaging: Secure channel to contact parents or answer student questions.

3. Student Dashboard

The Learning Hub

  • Academic Tracking: Clear display of GPA, latest grades, and attendance rates.
  • Scheduling: Current class timetables and events calendar.
  • Resources: Browse the digital library and access online learning materials.
  • Teacher Support: Message teachers directly for homework help or questions.

4. Parent Dashboard

The Progress Monitor

  • Child Monitoring: Switch between linked children to track specific grades and attendance.
  • Financial Transparency: View pending tuition fees, payment histories, and deadlines.
  • Parent-Teacher Communication: Instantly message teachers to discuss progress.
  • Awareness: Direct access to school-wide announcements and the academic calendar.

The Technology Stack

A modern MERN Architecture

  • Frontend: React.js, Vite, Tailwind CSS, Material Icons
  • Backend: Node.js, Express.js, Custom JWT Auth Middleware
  • Database: MongoDB Atlas (Cloud NoSQL) with optimized networking

Milestones Achieved 🏆

The complete journey from Day 1 to our Live Deployment

(Swipe down for details)

1. Foundation & Cloud Deployments

  • MERN Architecture: Initialized full-stack environment using Vite, React, Node.js, and Express.
  • Database Setup: Modeled MongoDB schemas for Users, Invoices, Messages, and Books.
  • Server-to-Client Config: Robust Render mapping, IPv4 database bypasses, and secure environment variable handling (VITE_API_URL).

2. Secure Authentication & Logic

  • Custom JWT Controls: Secured backend routes based on admin/teacher/parent/student roles.
  • Account Generator Engine: Automated creation of child accounts dynamically linked to parents via internal MongoDB referencing.
  • UI Password Display: Displaying system-generated passwords cleanly to bypass strict cloud SMTP limitations.

3. Premium Interface Engineering

  • Smart Dark Mode: Seamless Tailwind-powered toggle transforming the entire application aesthetically.
  • Dynamic Navigation: React-Router integrated sidebar adapting options precisely per user-role.
  • Autocomplete Selects: Replaced boring dropdowns with dynamic, searchable user data-lists.

4. Deep Role Modules

  • Admin & HR: Real-time overview of staff metrics, payroll processing, and student counts.
  • Finance Engine: Auto-generating, printable receipts and invoice transaction logic.
  • Academic Utilities: Digital library inventory mapping and E-learning file structure.
  • Grade Processing: Teacher dashboards built to submit multi-subject metrics to respective student accounts.

5. Global Communication

  • Direct Messaging Hub: Live messaging UI mimicking modern chat apps between all roles.
  • Persistent Localization: A powerful Context API dictionary rendering the entire app bilingual (FR/EN) instantly.
