Atlas Academy

Next-Generation School Management System

A unified cloud platform for modern education

The Problem

  • Schools struggle with fragmented legacy systems.
  • Communication gaps between teachers, parents, and admins.
  • Manual tracking for finances, attendance, and grading.

Our Solution

An all-in-one centralized platform that automates administration and unifies communication.

Accessible everywhere. Secure by design.

Role-Based Dashboards 👥

Four unique experiences connected to one secure database.

(Swipe down to explore roles)

1. Administrator Dashboard

The Command Center

  • Global Overview: Real-time statistics on enrollments, attendance, and staff counts.
  • Financial Control: Track tuition, operational expenses, and process teacher payroll.
  • User Management: Create accounts, assign roles, and handle HR leave requests.
  • Communication Hub: Broadcast urgent announcements across the platform.

2. Teacher Dashboard

The Academic Workspace

  • Classroom Management: Interactive timetables and digital daily attendance.
  • Grading & Exams: Upload results, assign grades, and update student progress dynamically.
  • Personal HR: Track monthly pay slips and officially request days off.
  • Direct Messaging: Secure channel to contact parents or answer student questions.

3. Student Dashboard

The Learning Hub

  • Academic Tracking: Clear display of GPA, latest grades, and attendance rates.
  • Scheduling: Current class timetables and events calendar.
  • Resources: Browse the digital library and access online learning materials.
  • Teacher Support: Message teachers directly for homework help or questions.

4. Parent Dashboard

The Progress Monitor

  • Child Monitoring: Switch between linked children to track specific grades and attendance.
  • Financial Transparency: View pending tuition fees, payment histories, and deadlines.
  • Parent-Teacher Communication: Instantly message teachers to discuss progress.
  • Awareness: Direct access to school-wide announcements and the academic calendar.

The Technology Stack

A modern MERN Architecture

  • Frontend: React.js, Vite, Tailwind CSS, Shadcn UI
  • Backend: Node.js, Express.js, JWT
  • Database: MongoDB Atlas (Cloud NoSQL)

Cloud Deployment ☁️

  • Client: Hosted on Vercel for edge delivery
  • Server: Hosted on Render for robust API handling
  • Data: Hosted on MongoDB Atlas securely
